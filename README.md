# About this repo
<br>
What is version control? (1-2 sentences)
<br>
Version control is the practice of tracking and managing changes in software code. It can be done locally, through the official version online, or a combination of both.
<br>
Why do developers use commits? (1-2 sentences)
<br>
Developers use commits because each change is permanently stored, either locally or in the main repository as a database. If anything goes wrong, developers can download and rollback to a previous version of the repository using that commit
